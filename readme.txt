-------------------------------------------------------
    Features
-------------------------------------------------------

Business_blog includes the following features:

    * Custom logo
    * Social media icons
    * Show full posts or excerpts
    * Custom excerpt length
    * Custom Read More button text
    * Make first post on blog extra wide
    * Hide/Show display controls
    * Custom CSS

Business_blog also has a premium plugin upgrade called Business_blog Pro that includes the following features:

    * Custom colors
    * Featured sliders
    * Featured videos
    * Additional layouts
    * Flexible header image
    * Google fonts
    * More widget areas
    * Display controls
    * Custom footer text

Business_blog Pro: https://www.competethemes.com/business_blog-pro/

-------------------------------------------------------
    Additional Info:
-------------------------------------------------------

    * All menus in Business_blog can have 1-3 tiers
    * Featured Images have a 2x1 ratio at all screen sizes
